## Normalized merged-feed item shape (used by every source fetcher and the merge module)

```javascript
// Each item in the merged feed:
{
  id: string,                    // stable per-source id, e.g. "dxnews:<url>", "dxworld:<guid>", "ng3k:<callsign>"
  title: string,                 // headline / first sentence
  description: string,           // up to ~200 chars of body
  url: string,                   // article / detail URL (used by D-13 click-navigate)
  publishDate: string,           // ISO 8601 UTC; for NG3K use startDate so sort works
  callsign: string | null,       // extracted via extractCallsign(); null = no dedup applied
  source: string,                // display name, exactly one of: "DXNEWS", "DX-WORLD", "NG3K"
  sourceUrl: string,             // homepage URL for D-12 click-label-navigate
  activityEndDate?: string,      // ISO 8601 UTC; NG3K only — drives D-02/D-05 freshness
}
```
